## 已知限制

1. **阶段检测歧义** — "利用漏洞" 中的 "漏洞" 先匹配 VULN_DISCOVERY，因为关键词遍历顺序问题。建议后续版本引入关键词权重或优先级机制
2. **Pydantic V2 警告** — `schema.py` 使用 class-based config 和 `model_fields` 实例访问，需迁移到 `ConfigDict` 和类属性访问
3. **MCP 服务器实际启动** — 当前测试不验证 MCP 服务器的进程级启停（需要外部依赖如 npx/Burp/Frida）
